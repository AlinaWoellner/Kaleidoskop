### Installation

1. Platform IO PLUGIN
   ```sh
   In der Suchleiste von VS Code eingeben und auf den installieren Button klicken
   ```
2. Projekt erstellen
   ```sh
   Im Plugin auf "import arduino Project" klicken und das geklonte Projekt auswählen
   ```
